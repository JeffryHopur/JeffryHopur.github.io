Windsheild Whiper System

Fully Functional Sonar

This project tries to replicate a cars windsheild whiper system. It is program using C and has 3 speeds slow, medium and fast and also has a windshield whiper fluid dispensing system.

It is controlled using the keypad and functions are programed in as 1 for slow 2 for medium and 3 for fast. Lastly there is also a full stop button as well as the fuild dispense button as it can be seen in the video
