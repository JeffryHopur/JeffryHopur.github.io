Sonar Detection With display

Fully Functional Sonar

This project consists of a servo motor, an ultrasonic sensor and an Arduino board. It utilizes the sensor to detect the object in front of it in real-time, and communicates the information to the Arduino.

The servo Motor allows for the spinning of the sensor's head allowing it to have the sonar feel. Lastly, the main component is the processing4 code allowing for the full Heads-Up display.
